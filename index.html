Tableau Colour Palette Converter

Have you ever wanted to make appealing colour palettes in Tableau? Coolors and Colorhunt are both great websites that can help with the colours that you want and this website is a tool to quickly convert into a Tableau custom palette for you.

Step 1: Choose your colours

Choose one of these options:

  • a) Go to coolors.co, then click on 'Start the generator' and choose the palette you like best. You can 'lock' certain colours as you go through the process. When you've got the colours that you like best, click on Export and then URL and return here.

  • b) Go to colorhunt.co, choose a palette and click on the 'link' button.

Step 2: Paste the link

Paste the link and enter the name for the palette in Tableau


Step 3: Click the button to continue

Step 4: Click the Copy button

Clicking the button will copy the XML into your clipboard

Step 5: Edit your preferences.tps file

OK, this is the penultimate step ...

Locate your preferences.tps file. The file is located at:
Documents\My Tableau Repository\Rreferences.tps

Edit the file with Notepad (or any other text editor) and paste your palette definition in place 

  <?xml version='1.0'?>
  <workbook>
  <preferences>

      PASTE YOUR PALETTE DEFINITION HERE

  </preferences>
  </workbook>

Indentation doesn't matter, but it can make it easier to read

You might have other palettes already defined, in that case be careful where you paste

Save the file

Step 6: Start Tableau Desktop and test

Fire up Tableau Desktop and check out your new palette
